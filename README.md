# AetherFlow Web

Vue 3 + TypeScript + Vite frontend for the AetherFlow project.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Axios
- Element Plus

## Local Development

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.development` before development.

```bash
cp .env.example .env.development
```

Default local proxy maps `/api` to `http://localhost:8080`.
